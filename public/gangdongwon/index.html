오답 이차방정식 / 인수분해

문제: 다음 이차방정식을 인수분해하시오.

x² + 5x + 6 = 0
나의 답: (x + 1)(x + 6) = 0
→ 두 수의 합이 5가 되지만 곱이 6이 아닌 조합을 선택 (1 × 6 = 6이지만 1 + 6 = 7 ≠ 5)
정답: (x + 2)(x + 3) = 0 → x = -2 또는 x = -3
→ 두 수의 합 = 5 (일차항 계수), 두 수의 곱 = 6 (상수항)

핵심 개념: 이차방정식의 인수분해

x² + (a+b)x + ab = (x+a)(x+b) 형태에서 합이 일차항 계수, 곱이 상수항이 되는 두 수를 찾는다.

풀이 A (십자곱):
x² + 5x + 6에서 합=5, 곱=6인 두 수 → 2와 3
∴ (x + 2)(x + 3) = 0
풀이 B (근의 공식):
x = (-5 ± √(25-24)) / 2 = (-5 ± 1) / 2
x = -2 또는 x = -3
유사 문제: x² - 7x + 12 = 0을 인수분해하시오.
힌트: 합=-7, 곱=12인 두 수를 찾으세요.

오답 원인 분석

두 수의 곱은 맞추었으나(1×6=6), 합 조건을 놓쳤다(1+6=7≠5). 인수분해 시 반드시 합과 곱 두 조건을 모두 확인해야 한다.

구분나의 답정답
두 수1, 6 2, 3
합7 ≠ 5 5
곱6 6

문제: 일차함수 y = 2x - 3의 그래프가 x축과 만나는 점의 좌표를 구하시오.

y = 2x - 3, x축과의 교점 = ?
나의 답: (0, -3)
→ x축과의 교점이 아닌 y절편을 구함 (x=0을 대입한 결과)
정답: (3/2, 0)
→ x축과의 교점에서는 y = 0. 0 = 2x - 3 → x = 3/2

핵심 개념: 일차함수의 그래프와 절편

y = ax + b에서 x절편은 y=0일 때, y절편은 x=0일 때의 값이다.

절편구하는 방법이 문제
x절편y = 0 대입0 = 2x - 3 → x = 3/2
y절편x = 0 대입y = 2(0) - 3 = -3

오답 원인 분석

"x축과 만나는 점"은 y = 0을 대입해야 하는데, 반대로 x = 0을 대입하여 y절편을 구했다. x절편과 y절편의 차이를 정확히 구분해야 한다.

구분나의 답정답
대입한 값x = 0 y = 0
구한 좌표(0, -3) = y절편 (3/2, 0) = x절편
